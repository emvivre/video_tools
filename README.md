# video_tools
Shell scripts to manipulate videos, depends on ffmpeg.

Description of each script : 
- video_concat.sh : concatenate several video to generate a single video
- video_extract_frames.sh : extract each images frames 
- video_gen.sh : generate a video from several images frames
- video_get_fps.sh : get FPS of the input video
- video_get_nb_frame.sh : get the number of frames of the input video
- video_reset_start_time.sh : reset the start_time property of a video
- video_slice.sh : slice the input video into several video parts of same duration.
- video_truncate.sh : extract a part of the input video
